# Welcome to Gambit

I welcome all the users to this minimalistic chess app.

## Tech Stack

I've used React and it's ecosystem to develope the frontend and for the backend I've used Fastify and Mongo DB to handle http request from the clients.

## Description

This project is built by trying to learn fastify. Fastify is a highly performant, low-overhead web framework for Node.js, designed specifically to maximize developer efficiency and request handling speed.

### What I've learned

During the development, I've found that fastify handles request faster and better than Express. It's really fast and I'm not kidding about it. Since I am an intermediate developer, learning fastify was bit difficult at first but after I got the basics it was smooth after that. The docs are hard to navigate too, I had to spend more time trying to figure out the docs then the actual code😂.

I also am greatful for claude in helping me through the journey. Without it I would have been stuck for months trying to get the UI right.
